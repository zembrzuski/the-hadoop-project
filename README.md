Como utilizar esse projeto
--------------------------------------------------------------------------------------------


- Criar as VMS
- Copiar os ips gerados para o inventory
- Provisionar o master
- Provisionar os slaves
- formatar os namenodes:
  $HADOOP/bin/hadoop namenode -format
